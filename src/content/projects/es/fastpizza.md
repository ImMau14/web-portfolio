# Aplicación de FastPizza

Esta aplicación de escritorio está diseñada para la gestión integral de ventas. Permite el cálculo del monto total de las ventas diarias, ofrece la tasa de cambio del dólar en tiempo real, genera informes en formato PDF y facilita la gestión de la base de datos, entre otras funcionalidades clave.

Actualmente, el proyecto se encuentra en fase de prototipo. Se prevé el desarrollo de una nueva versión que incorporará mejoras y correcciones.

> Este proyecto representa mi primera experiencia profesional como programador. Mi rol abarca la definición de las tecnologías a emplear, el diseño UX/UI, y una parte del desarrollo del _backend_ y la gestión de la base de datos.
