# Website de TScrapingBot

Este proyecto consiste en el desarrollo del sitio web oficial de TScrapingBot, un bot de Telegram. El sitio incorpora animaciones dinámicas y soporte para múltiples idiomas, ofreciendo una experiencia de usuario enriquecida.

> Fue concebido como una oportunidad para adentrarme en Astro y TypeScript. El objetivo principal de este desarrollo fue adquirir experiencia práctica y consolidar conocimientos en ambas tecnologías, aplicándolos en un proyecto web moderno.
